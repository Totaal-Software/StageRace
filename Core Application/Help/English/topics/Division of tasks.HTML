Division of Tasks

It is a good thing to make a division into the following tasks, and also in the moment on which these tasks are executed:

  • Configuring of an event
    (stage race organization)
    This can be done in the days preceding the event, in cooperation with an organization representative.
  • Utilization of an event
    (race administration)
    This takes place during the various stages, in cooperation with the members of the jury which are present.

You can enforce a division of tasks by locking the configuration of the event through the access settings. That way you can prevent administrators that handle the race from changing the race's configuration.
